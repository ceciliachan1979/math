\subsubsection*{Exercise 1.1.1. (Gapry)}
We do the inference:
\begin{flalign*}
                                x &< 0  \text{ (Assumption)}                          &\\
                               -x &> 0  \text{ (Proposition 1.1.8. (i)) $\cdots$(1) } &\\
                                y &< z  \text{ (Assumption)}                          &\\
                            y - z &< 0                                                &\\
                         -(y - z) &> 0  \text{ (Proposition 1.1.8. (i)) $\cdots$(2) } &\\
(1) \cdot (2) = -x \cdot -(y - z) &> 0  \text{ (Definition 1.1.7. (ii))}              &\\
                 x \cdot  (y - z) &> 0                                                &\\
                          xy - xz &> 0  \text{ (Distributive law)}                    &\\
                               xy &> xz \text{ (Q.E.D.) }
\end{flalign*}
