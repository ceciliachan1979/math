\subsection*{Dummit Exercise 1.1.25 (Cecilia)}

For all $p$ and $q$ in the group:

\begin{align*}
  pqpq &= 1   && \text{(apply } x^2 = 1 \text{ for } x = pq \text{)} \\
  ppqpq &= p  && \text{(left multiply } p \text{ on both sides)} \\
  qpq &= p    && \text{(since } pp = 1 \text{)} \\
  qpqq &= pq  && \text{(right multiply } q \text{ on both sides)} \\
  qp &= pq    && \text{(since } qq = 1 \text{)}
\end{align*}

Therefore the group is Abelian. \qed
